# 隐私政策

语言：[English](./PRIVACY.md) | 简体中文

最后更新：2026 年 5 月 8 日

本隐私政策说明 Mono Dash（以下简称“Mono Dash”“本 App”“我们”）在你使用本 App 时如何处理信息。

Mono Dash 是面向 1Panel 的第三方移动管理客户端。Mono Dash 不是 1Panel 官方产品，也不隶属于 1Panel。

本文档用于公开透明说明，也可作为 GitHub、App Store、Google Play 等平台使用的公开隐私政策链接。本文档不构成法律意见。

## 概要

Mono Dash 不运营任何用于用户账户、数据分析、行为跟踪、日志收集或 1Panel 数据处理的后端服务器。我们不收集、不记录、也不分析你的 App 内行为。

你的服务器列表、App 设置和 1Panel 连接信息保存在你的设备本地。你的所有 1Panel 操作均由你的设备发起，并直接连接到你自己配置的 1Panel 服务器。

如果当前版本启用了应用内购买功能，Apple、Google Play 和 RevenueCat 可能会用于处理购买和同步购买状态。这些购买服务与 1Panel 服务器数据相互独立。

## 无分析和跟踪

Mono Dash 不包含由 Mono Dash 运营的分析服务、跟踪 SDK、广告 SDK、行为日志后端或崩溃日志收集后端。

我们不会记录你添加了哪些面板、打开了哪些页面、执行了哪些操作、运行了哪些命令、查看了哪些文件，或你使用本 App 的频率。

## 本 App 可能处理的信息

根据你的使用方式，Mono Dash 可能会在你的设备本地访问或保存以下信息，或将这些信息从你的设备直接发送到你配置的 1Panel 服务器。我们不会在自己的服务器上接收这些信息。

### 面板连接信息

当你添加 1Panel 服务器时，本 App 可能会保存：

- 服务器名称或显示名称
- 主机名或 IP 地址
- 端口
- HTTP 或 HTTPS 设置
- 是否允许不安全连接或自签名证书连接
- 创建时间、最后使用时间和本地排序信息
- 你自行配置的自定义请求 Header

这些信息保存在你的设备本地，用于让本 App 重新连接你的面板。它们不会上传到 Mono Dash 后端。

### API Key 和认证数据

本 App 会使用操作系统提供的安全存储能力在本地保存 1Panel API Key（如系统支持）。本 App 使用 API Key 在你的设备本地签名请求，并发送给你配置的 1Panel 服务器。

Mono Dash 不会将你的 1Panel API Key 发送到任何 Mono Dash 后端。

### 服务器和面板数据

当你连接到 1Panel 实例时，本 App 可能会在你的设备和该 1Panel 服务器之间展示或传输数据，包括：

- 服务器状态、指标、进程信息、日志和 SSH 信息
- 文件名、目录路径、文件内容，以及上传或下载的文件
- 网站、运行环境、数据库、备份、防火墙、计划任务、应用、容器和系统设置
- 使用终端功能时的终端输入和输出
- 面板返回的操作结果和错误信息

这些数据仅在你的设备和你配置的 1Panel 服务器之间直接交换。Mono Dash 不提供也不控制该服务器，我们也不会收集这些数据。

### 文件、照片和文档

如果你选择上传文件、导入文件、下载文件、分享导出的日志或查看图片，本 App 可能会访问你在设备上选择的文件或照片。本 App 仅为完成你请求的操作而使用这些访问权限。

你选择上传的文件可能会被发送到你选择的 1Panel 服务器。从 1Panel 服务器下载的文件可能会保存在你的设备本地。

### 局域网访问

在需要系统授权的平台上，本 App 可能会请求局域网访问权限，以便连接你局域网内的 1Panel 服务和网站资源。

### 购买信息

如果当前版本启用了应用内购买，购买将由 App Store 或 Google Play 处理。Mono Dash 可能会使用 RevenueCat 加载商品、处理购买流程、恢复购买和同步权益状态。

购买相关信息可能包括购买状态、商品标识、权益标识、收据、交易信息、购买服务使用的设备或 App 标识，以及相关诊断信息。支付卡信息由 Apple 或 Google 处理，Mono Dash 不会收到支付卡信息。购买服务不会从 Mono Dash 收到你的 1Panel 服务器数据。

更多信息请查看相关服务的隐私政策：

- Apple 隐私政策：https://www.apple.com/legal/privacy/
- Google 隐私权政策：https://policies.google.com/privacy
- RevenueCat 隐私政策：https://www.revenuecat.com/privacy

### 设备和 App 信息

本 App 可能会向你配置的 1Panel 服务器发送 User-Agent 字符串，其中可能包含 App 名称、App 版本、平台和操作系统版本。这用于帮助服务器识别来自本 App 的请求。

### 日志和诊断

本 App 运行时可能会产生本地调试或诊断日志。这些日志保存在你的设备上，除非你主动选择分享它们。本 App 包含一定的脱敏保护，以降低敏感 Token 和密码被意外写入日志的风险；但如果日志中可能包含服务器地址、路径、运维数据或其他敏感信息，请不要公开分享。

## 本 App 如何处理信息

Mono Dash 在你的设备本地处理信息，用于：

- 连接和管理你配置的 1Panel 服务器
- 保存你的本地 App 设置和服务器列表
- 对发送给你配置的 1Panel 服务器的请求进行认证
- 在你请求时上传、下载、查看、编辑或分享文件
- 展示服务器状态、日志、终端会话、网站、数据库、容器和其他 1Panel 资源
- 处理、验证、恢复和同步购买状态


## 信息如何离开你的设备

Mono Dash 不出售你的个人信息，也不会将你的 1Panel 数据分享给 Mono Dash 后端。

信息仅可能在以下情况下离开你的设备：

- 在你执行相关操作时，发送到你配置的 1Panel 服务器
- 如果启用了购买功能，发送到 Apple、Google Play 和 RevenueCat，用于处理购买和同步购买状态
- 当你通过系统分享功能主动选择分享给其他 App 或服务时

由于我们不运营用于 App 使用数据的后端，我们通常没有用户操作记录、1Panel 数据、API Key、文件或分析数据可供披露。

## 本地存储和安全

Mono Dash 将面板连接数据和 App 偏好设置保存在你的设备本地。API Key 会尽可能使用系统安全存储。我们不会将这些本地数据复制到 Mono Dash 后端。

你需要自行保护设备、1Panel 服务器和 API Key 的访问安全。建议尽可能使用 HTTPS，并且只在你信任的服务器上启用不安全连接或自签名证书连接。

任何存储或传输方式都无法保证绝对安全。由于 Mono Dash 直接连接你配置的服务器，你的数据安全也取决于你的服务器配置、网络环境、证书设置和 1Panel 访问控制。

## 数据保留

Mono Dash 会将本地保存的信息保留在你的设备上，直到你删除它、从 App 中移除服务器、清除本地数据、卸载 App，或操作系统移除这些数据。

下载的文件会保留在你的设备上，直到你删除它们。存储在 1Panel 服务器上的文件和日志由该服务器及你自己的服务器保留策略控制。

购买记录可能会由 Apple、Google Play、RevenueCat 和相关支付服务根据其政策和法律义务保留。

## 删除你的数据

你可以通过从 App 中移除服务器来删除本地保存的面板连接信息。这也会删除与该服务器关联的本地 API Key。

你可以通过 App 的下载管理功能或操作系统的文件管理工具删除设备上的已下载文件。

如需删除存储在 1Panel 服务器上的数据，请使用 1Panel、Mono Dash 中作用于该服务器的功能，或你的服务器管理工具。Mono Dash 不控制你运营或连接的 1Panel 实例的服务端数据保留。

如需处理购买相关的删除或隐私请求，请联系 Apple、Google 或 RevenueCat。

## 第三方服务

Mono Dash 可能会与以下第三方服务或组件交互：

- 你自行配置的 1Panel 服务器
- 用于购买和分发的 Apple App Store 和 Google Play
- 用于购买权益管理的 RevenueCat
- 操作系统服务，例如安全存储、文件选择器、分享、WebView 和局域网权限提示

这些服务可能会根据各自的隐私政策处理数据。

## 儿童隐私

Mono Dash 面向服务器管理场景，并非面向儿童。我们不会有意收集儿童的个人信息。

## 跨地区使用

你可以使用 Mono Dash 连接位于不同国家或地区的服务器。发送到你配置的 1Panel 服务器的信息将在该服务器所在位置被处理。
